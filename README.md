## Проекты

Здесь представлены мои проекты, демонстрирующие навыки анализа данных и решения бизнес-задач.

### 1. Анализ воронки и A/B-тестирование (сервис аренды авто МобиКар)
*   **Задача:** Компания столкнулась с падением конверсии из установки в первую поездку на 15% за 2 месяца. Требовалось найти причину.
*   **Решение:**
    *   Подготовила данные с помощью SQL, построил воронку из 6 этапов (Установки → Регистрация → Поиск → Просмотр → Бронь → Поездка) за 8 недель.
    *   Визуализировала динамику конверсий в Power BI, выделив зоны падения.
    *   Выявила критическую точку: падение конверсии на этапе **«Регистрация → Открыли поиск»** с ~70% до 50% (-20 п.п.).
    *   Сформулировала **3 гипотезы** для A/B-теста и предложила дизайн эксперимента.
*   **Инструменты:** SQL, Power BI, Excel, Python (для статистической проверки гипотез).
*   **Результат:** Локализовала проблему, подготовил рекомендации продуктовой команде.

### 2. Когортный анализ удержания пользователей (Retention)
*   **Задача:** Понять, на каком этапе пользователи мобильного приложения перестают возвращаться, и предложить улучшения.
*   **Решение:**
    *   Провела валидацию событийных данных, проверив их на полноту и корректность.
    *   Построила когортный анализ Retention на D1, D7, D30.
    *   Визуализировала воронку активации (Запуск → Регистрация → Первое действие).
    *   Выявила аномалию: 40% пользователей не доходят до первого действия после регистрации.
*   **Инструменты:** Python (pandas, matplotlib, seaborn), Excel.
*   **Результат:** Предложила упростить онбординг и добавить push-напоминания на 1-й день для повышения Retention.
