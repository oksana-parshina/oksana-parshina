# Анализ эффективности маркетинга автодилера (BMW & Mercedes)
Пилотный проект для автодилера: объединение данных Google Analytics и CRM для оценки реальной эффективности рекламы — от клика до продажи.

# Бизнес-задача
Автодилер тратит бюджет на интернет-рекламу, но не знает:
- Какие маркетинговые кампании работают лучше всего?
- Какие модели продаются лучше всего через интернет?

# Цель проекта
Создать дашборд, который покажет полную картину.

# Стек
- Power BI (визуализация, DAX)
- Power Query (ETL, объединение источников)
- Google Analytics + CRM (источники данных)
- Client ID (сквозная аналитика)

# Ключевые инсайты
1. Кампания "36451549" приводит к самому высокому отклику.
2. Модель Mercedes e220 лучше всего продаётся в онлайне.

# Что я сделала
- Объединила данные из 2 источников через Client ID
- Построила модель данных (звезда)
- Создала интерактивный дашборд для руководителя
